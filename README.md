# MeowMeowCare 貓咪日常紀錄

新手只要跟下面步驟做，就可以在瀏覽器打開 App，**唔使改任何 code**。

## 第一步：打開終端機

- Mac：按 `Command + 空白鍵`，搜尋 **Terminal（終端機）**，按 Enter

## 第二步：進入專案資料夾

複製下面整行，貼到終端機，按 Enter：

```bash
cd /Users/sonialam/my-cat-app
```

## 第三步：安裝套件（第一次先要做）

```bash
npm install
```

## 第四步：啟動 App

```bash
npm run dev
```

終端機會顯示類似：

```
  ➜  Local:   http://localhost:5173/
```

用 Chrome / Safari 打開 **http://localhost:5173/** 即可。

> 要停止 App：在終端機按 `Ctrl + C`

## 功能簡介

### 🏠 首頁
- **今日餵食**：熱量 vs 建議值
- **今日總水分**：飲水 + 乾糧/濕糧水分
- **⚠️ 水分不足警號**：低於建議會顯示紅色提示
- **水分攝取計算**：詳細分拆

### 🍽️ 飲食分頁
- 新增每筆餵食（時間、乾糧/濕食/零食、克數、熱量）
- 記錄飲水量（+20ml 或自訂）
- 選日期查看該日是否達標

### 底部 ＋
- 快速新增餵食紀錄

### ⚙️ 設定
- 填寫貓貓**體重 (kg)**（水分建議靠呢個計算）

---

## 水分點樣計？

| 項目 | 計法 |
|------|------|
| 建議總水分 | 體重 × **40～60** ml（例：4kg → 160～240 ml）|
| 乾糧水分 | 乾糧克數 × **10%** |
| 濕糧水分 | 濕糧克數 × **75%** |
| 今日總水分 | 碗裡飲水 + 乾糧水分 + 濕糧水分 |

低於建議下限約 85% 會出 **⚠️ 水分不足警號**。

## 常見問題

**Q：畫面全白或好怪？**  
在瀏覽器按 F12 → Console，或清除舊資料後重新整理：

```javascript
['meow_cats_list_v30','meow_cats_list_v29'].forEach(k => localStorage.removeItem(k))
location.reload()
```

**Q：我改咗 code 點保存？**  
所有程式主要在 `src/App.vue`，用 Cursor 保存後，重新整理瀏覽器即可（`npm run dev` 要繼續開住）。
